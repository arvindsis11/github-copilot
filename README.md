# Github Copilot AI Agent (Learning Project)

This is an extensible Spring Boot project designed to learn AI integration step by step.

## Features (Initial Version)
- Simple AI Agent API
- Calls LLM (OpenAI compatible)
- Clean architecture for extensibility

## API

POST /agent/ask

Request:

{
  "message": "Explain microservices"
}
